# Ly Nguyen Personal Website

This is a simple static personal academic website with four main sections:

1. About Me
2. Research
3. Teaching
4. Publications

## Files

- `index.html` contains the website content.
- `style.css` controls the design.
- `images/` contains the visual graphics used throughout the website.

## How to add your own photo

1. Add your professional headshot to the `images` folder.
2. Rename it as `profile.jpg`.
3. Open `index.html`.
4. Replace this line:

```html
<img src="images/profile-placeholder.svg" alt="Professional profile placeholder" />
```

with:

```html
<img src="images/profile.jpg" alt="Ly Nguyen professional headshot" />
```

## How to publish

You can upload this folder to GitHub Pages, your FAMU faculty web space, Netlify, or another static website host.
